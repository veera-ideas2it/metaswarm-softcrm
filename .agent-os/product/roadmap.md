# Product Roadmap
# SoftCRM

## v1 — Minimal Working CRM (Current Build)
Goal: A fully usable CRM that a software sales team can use from day one.

### Modules in v1
- [x] Auth (login, logout, JWT refresh, roles)
- [x] Pipeline Kanban board (drag-drop, all stages)
- [x] Deal management (create, edit, move stage, soft delete)
- [x] Contact management (linked to companies and deals)
- [x] Company/account management
- [x] Activity logging (calls, emails, meetings, notes, tasks)
- [x] Dashboard (KPIs, funnel chart, recent activity)
- [x] Basic reports (pipeline by stage, won/lost)
- [x] Team management (invite, roles, deactivate)

### Out of Scope for v1
- Email integration (send/receive email from CRM)
- Calendar sync (Google/Outlook)
- Document e-signing
- Custom fields
- Webhook / API for external integrations
- Mobile app

## v2 — Productivity (Next)
- Email integration (log sent emails automatically)
- Google Calendar sync for meetings
- Deal templates (clone a deal with prefilled info)
- Custom pipeline stages (configurable per team)
- CSV import for contacts/companies
- Advanced reports and date-range filters

## v3 — Intelligence (Future)
- Deal health score (AI-generated based on activity)
- Forecasting with confidence intervals
- Automated activity suggestions ("No contact in 14 days")
- Integration marketplace (Slack, HubSpot, Salesforce migration tool)
